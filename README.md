## 🗻 做一個會員, 錢包的 API Service
* 👤 會員
  * C -> 註冊, 登入
  * R -> 查詢會員資料
  * U -> 更新資料
  * D -> 停用, 凍結
* 💰 錢包
  * C -> 開戶
  * R -> 查詢餘額, 交易紀錄
  * U -> 存款, 提款, 轉帳
  * D -> 停用, 凍結
